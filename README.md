# DuoFolio

DuoFolio was made for duos, with all the pros. A duo project initiated by two computer science students seeking to make portfolio sharing simpler, and more enjoyable.

## Features

- Feature Beta v1: General Header Navigation Bar is at the top, with underlined and highlighted meaning currently selected page. Responsive Design is still
under works

## Prerequisites

Ensure you have a local or private server ready to host on.

## Installation

Here's the step-by-step instructions to get the local development environment running:

```bash
# Clone the repository
git clone https://github.com/Git-67/UXUI_Assignment.git

# Navigate into the directory
cd UXUI_Assignment
```

Afterwards run the folder on your server.
